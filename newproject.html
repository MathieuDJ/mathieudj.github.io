Projecten

Slimme Quiz

Voor System Engineering & Analysis moesten wij voor Alexander een Slimme Quiz maken. Alexander gebruikte eerst papieren, powerpoints... en hiervoor was veel personeel en werk nodig. Het was belangrijk dat wij een systeem zouden ontwikkelen dat veel zou automatiseren en een systeem waar alles mee geregeld kan worden. Zo is er minder personeel nodig en kunnen men sneller en eenvoudiger quizzen opstellen.

In dit project hebben wij dus met een Raspberry Pi 4 en wat elektronica een quiz-systeem gemaakt. Wij hebben gebruik gemaakt van een API en Database. Als men op een knop duwt wordt dat antwoord naar de database gestuurd en verbeterd. Er wordt dan een score toegevoegd. Wij hebben voor Alexander dus echt een heel systeem opgesteld dat kan gebruikt worden om te quizzen. Er is een bakje met knoppen voor de quizzers, een interface waar alles op wordt getoond en waar men heel de quiz met kan opstellen.

In dit project hebben we geleerd om een klant zijn behoeften te realiseren. Wij hebben in team gewerkt en dus onze "team" skills verder ontwikkelt maar ook onze technische skills. We moesten echt luisteren naar de klant om een systeem op maat te maken en hebben we onze soft skills ook veel moeten gebruiken

Ik heb mij in dit project vooral bezig gehouden met de elektronica. Ik heb er dus voor gezorgd dat er een fysiek bakje was met knoppen dat de quizzers kunnen gebruiken en dat de informatie correct naar de API en database wordt verstuurd.

Energiemeter

Voor IoT Advanced moesten wij een digitale en analoge meter uitlezen. Deze data moesten wij dan naar een database versturen en op basis van deze database werd en dan een heel dashboard gemaakt. We hadden ook de taak om het piektarief te berekenen. Dit project kwam eigenlijk op het juiste moment aangezien ze rond deze tijd ook je capaciteitstarief berekenen op basis van je hoogste kwartierpiek.

In dit project hebben wij zelf een P1 interface kabel gemaakt om de digitale meter uit te lezen. Deze hebben wij gekoppeld aan een Pi zodat de data kon worden uitgelezen. Wij hebben er ook voor gezorgd dat de waardes in een database worden opgeslagen en dat er een dashboard van wordt gemaakt. De waardes zoals kwartierpiek werden op een LCD geprint.

Met dit project konden wij dus een energiemeter uitlezen en er een dashboard van maken zodat mensen hun energieverbruik in de gaten kunnen houden.

In dit project heb ik het meeste werk gedaan omdat ik ook de digitale meter thuis had. Ik heb de P1 interface kabel gemaakt zodat wij de data konden uitlezen. Eens de data was uitgelezen heb ik er voor gezorgd dat deze in een database werd opgeslagen en dat er een dashboard van werd gemaakt. Ik heb mij in dit project ook bezig gehouden met de connecties van alles.

In dit project heb ik vooral mijn technische skills versterkt in de IoT. Er kwam veel elektronica aan te pas en ik heb veel moeten opzoeken en bijgeleerd.

Dit project was in team dus softskills kwamen er ook aan te pas. Zo heb ik mijn communicatievaardigheden versterkt want het was toch wel belangrijk om met je team te communiceren en af te spreken.

Pompoen Serre Automatiseren

Voor Project IoT moeten wij een pompoen serre automatiseren. Paul Boonen groeit pompoenen voor het wereldrecord en zou het fijn vinden als hij met een app vanop afstand zijn pompoen verder kan laten groeien door bijvoorbeeld water te geven, de ramen open te kunnen zetten... Dit project is nog niet af maar wij moeten het water, de ventilatie etc. allemaal automatiseren en hij moet de pompoenen op een app kunnen zien en alles met een app kunnen besturen.

Wij werken met wifi en hebben er al voor gezorgd dat de elektronica met wifi kan communiceren. Wij hebben ook al een duidelijk plan van aanpak samengesteld. In de serre zelf hebben wij nog niet gewerkt. Dit gebeurt tijdens de projectweek.

Dit project is een groepswerk en ik heb mij bezig gehouden met opzoekwerk rond de wifi modules die wij gebruiken. Het project is nog zeker niet af maar ik heb er al heel wat uit kunnen leren. Zo weet ik nu hoe je met Wifi, elektronica kan besturen. Ik heb ook al een vergadering moeten leiden en een powerpointpresentatie gegeven. Hieruit heb ik nuttige feedback gekregen dat ik bij andere vakken en presentaties kan gebruiken.

Chatbots

In mijn vrije tijd ben ik bezig met een chatbot te maken. Dit project doe ik uit interesse en is voor mijzelf. AI en automatisering interesseren mij heel erg en zo ben ik eigenlijk op dit project gekomen. Een chatbot kan gebruikt worden voor heel wat verschillende toepassingen. Je kan er met chatten, data laten halen van het internet en op basis van deze data beslissingen laten maken. Je kan ook je klantensupport automatiseren...

Op dit moment heb ik een chatbot die ik aan het trainen ben met een dataset van ongeveer 30GB. Ik doe dit omdat ik graag eerst wil leren hoe een chatbot werkt, hoe je er eentje traint, welke programma's je gebruikt om er één te maken...

Ik heb dus al een chatbot waar ik mee kan praten. Hij kan nog nergens geïmplementeerd worden.

In dit project heb ik vooral mijn technische skills zoals python, opzoeken op het internet, technische problemen oplossen... bijgewerkt. Het is een individueel project dus veel softskills heb ik nog niet moeten toepassen. Als ik beter wordt in het maken van chatbots en echt toepassingen kan vinden dan wil ik zeker proberen om er te verkopen.
